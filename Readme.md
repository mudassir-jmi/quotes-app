# QuoteSphere 🌟

A beautiful, fully responsive quotes listing web application built with HTML, Tailwind CSS, and vanilla JavaScript. Fetches inspirational quotes from the FreeAPI Quotes endpoint.

## ✨ Features

- **Browse Quotes** - Elegant card grid layout with pagination
- **Search** - Real-time search by quote or author
- **Random Quote** - Surprise yourself with a random quote in a modal
- **Copy Quotes** - One-click copy for single quotes or all quotes
- **Responsive Design** - Works perfectly on mobile, tablet, and desktop
- **Modern UI** - Dark theme with smooth animations and hover effects
- **Keyboard Shortcut** - Press `/` to focus search
